# LP3 - SAMPLE REALTY | 法人向けオフィス賃貸LP

法人専門のオフィス仲介サービスの集客特化型ランディングページ。

## デザインコンセプト
- **テーマ**: ラグジュアリー・プロフェッショナル
- **カラー**: リッチブラック × ゴールド × クリーム
- **画像**: Unsplashフリー素材（著作権クリア）

## 技術スタック
- Next.js 16 + TypeScript
- Tailwind CSS 4
- スクロールアニメーション（IntersectionObserver）
- パララックス効果
- カウントアップアニメーション

## セットアップ
```bash
npm install
npm run dev
```

## セクション構成
1. ファーストビュー（カウントアップ付きヒーロー）
2. お悩み共感セクション

4. 取り扱い物件タイプ（画像付き）
5. ご利用の流れ（パララックス）
6. 導入事例（Before/After）
