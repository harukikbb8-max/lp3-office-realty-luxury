# LP3 要件定義書：法人・オフィス賃貸向け 集客LP（ラグジュアリー版）

---

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | 法人・オフィス賃貸向け 集客特化型LP |
| クライアント業種 | 不動産会社（法人向けオフィス賃貸） |
| 制作ツール | デザイン：Figma / 実装：Wix（Wixエディタ） |

## 2. デザイン強化ポイント（ラグジュアリー版）

- ダークテーマベース（ブラック × ゴールド × ホワイト）
- フリー素材画像（Unsplash）による高品質ビジュアル
- スクロールアニメーション全面適用
- パララックス効果
- ガラスモーフィズム要素
- ゴールドのアクセントカラーを統一

### カラーパレット
| 用途 | カラー |
|------|--------|
| メインカラー | #0A0A0A（リッチブラック） |
| セカンダリ | #1B2A4A（ダークネイビー） |
| アクセント | #C5A572（ゴールド） |
| CTA | #C5A572（ゴールド）/ #2563EB（ブルー） |
| 背景 | #FAF8F5（クリーム） |
| テキスト | #FFFFFF / #1A1A1A |

### 画像素材
- **すべてUnsplash等のフリー素材を使用（著作権クリア）**
- モダンなオフィス空間
- ビジネス街・高層ビル群
- ミーティングルーム
- コワーキングスペース
